# Assembly-kode for nokre eksempelprogram

## Fibonacci

```
0: sett r0 0
1: sett r1 1
2: pluss r2 r0 r1
3: vis r2
4: flytt r0 r1
5: flytt r1 r2
6: sett r3 2
7: sett r4 0
8: grein r3 r4
```

## Gjettelek

```
0: -
1: tilfeldig r0
2: brukartal r1
3: minus r2 r0 r1
4: sett r3 2
5: grein r3 r2
6: minus r2 r0 r1
7: grein r3 r2
8: fyrverkeri
```
